# astrbot_plugin_tataru

TataruBot2 的 AstrBot 插件迁移版本。

当前已迁移的低成本功能：

- `帮帮忙`
- `暖暖`
- `选门`
- `仙人彩`
- `日历`
- `攻略`
- `招募`
- `抽卡`

将 `astrbot_plugin_tataru` 目录放入 AstrBot 的 `data/plugins` 目录后，在 AstrBot WebUI 重载插件即可测试。

## 招募

- `招募 陆行鸟`：默认返回最近 10 条。
- `招募 陆行鸟 随机任务`：按分类筛选。
- `招募 陆行鸟 随机任务 20`：按分类筛选并指定数量，最多 40 条。
- `招募 陆行鸟 高难度任务`：`高难任务`、`高难度任务`、`高难本` 等别名都会匹配高难任务分类。
- 支持分类：随机任务、迷宫挑战、行会令、讨伐歼灭战、大型任务、高难任务、PvP、金碟、危命任务、寻宝、狩猎、采集、深层迷宫、特殊场景探索、多变迷宫、其他。
- 数据源优先使用 remote-party-finder v1 API，失败时自动回退到网页解析。

## 配置

- `默认使用国际服日历`：关闭时默认查询国服日历，开启后默认查询国际服日历。
- 命令也支持临时指定服务器：`日历 国服`、`日历 国际服`。
