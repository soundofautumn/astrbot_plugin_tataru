# 更新日志

## v0.12.2

### 优化

- `看看微博` 增加 `weibo.com/ajax/statuses/mymblog` 网页端接口作为 Cookie 场景下的备用数据源。
- 当微博移动端接口无数据时，`看看微博` 会使用网页端接口补足最新 5 条微博。

## v0.12.1

### 优化

- 参考 `astrbot_plugin_weibo_monitor` 优化 `看看微博` 请求头和微博卡片过滤逻辑。
- `看看微博` 现在仅处理 `card_type == 9` 的真实微博卡片，并更严格跳过置顶微博。

## v0.12.0

### 新增

- 新增 `看看微博` 命令，获取 FF14 官方微博最新 5 条非置顶微博。
- 新增 `weibo_cookie` 插件配置项，可填写微博 Cookie 以提高微博移动端接口访问稳定性。

## v0.11.2

### 优化

- `房子` 默认只返回前 10 条空房，避免一次发送过多内容。
- `房子` 支持按房型、房区和房号组合筛选，例如 `房子 银泪湖 森都一区 S`、`房子 银泪湖 森都一区5号`、`房子 银泪湖 森都 5号`。

## v0.11.1

### 修复

- 拆分 `房子` 和 `房屋` 命令处理器，避免 AstrBot 未正确注册叠加装饰器导致消息落入模型对话。

## v0.11.0

### 新增

- 新增 `房子`/`房屋` 命令，支持按服务器、主城和房屋大小查询空房。
- `房子` 优先使用 `house.ffxiv.cyou` 房屋 API，并保留原 TataruBot2 的服务器、主城和房型参数格式。

## v0.10.1

### 文档

- 重写 README，补充当前已实现命令、示例、返回形式、配置项和使用的数据源/项目列表。

## v0.10.0

### 新增

- 新增 `价格` 命令，默认返回 10 条市场挂单。
- `价格` 支持 `HQ`、四大区和具体服务器作为筛选项，例如 `价格 红玉海 铁矿 HQ 20`。
- `价格` 使用 Universalis 查询物价，物品 ID 搜索复用 XIVAPI v2。

## v0.9.0

### 新增

- 新增 `物品` 命令，支持查询物品基础信息、图标和获取方式。
- `物品` 详情、图标和来源数据直接使用 Garland Tools 国服站，名称搜索使用 XIVAPI v2 辅助匹配物品 ID。

## v0.8.0

### 新增

- `招募` 支持按具体服务器查询，例如 `招募 红玉海`。
- `招募` 支持按职业名查询，例如 `招募 战士`，未指定大区或服务器时会查询全服。

## v0.7.1

### 修复

- 修复 `招募` 使用 v2 API 时无法按 XIVAPI 解析后的副本名搜索的问题，例如 `招募 陆行鸟 高难任务 光暗未来绝境战`。

## v0.7.0

### 优化

- `招募` 数据源切换为优先使用 remote-party-finder v2 API。
- `招募` 通过 XIVAPI v2 批量解析 v2 API 返回的世界 ID 和副本 ID，保持输出可读。
- `招募` 保留 v1 API 和网页解析作为兜底，降低外部接口波动影响。

## v0.6.2

### 新增

- `招募` 命令在参数无法匹配分类别名时，会自动作为关键词调用招募 API 的 `search` 参数。
- `招募` 支持同时使用分类、关键词和数量，例如 `招募 陆行鸟 高难任务 绝龙诗 20`。

## v0.6.1

### 修复

- 修复 `招募 陆行鸟 高难度任务` 未识别分类别名，导致回退成全部分类的问题。

## v0.6.0

### 优化

- `招募` 数据源改为优先使用 remote-party-finder v1 API，减少网页结构变化导致的解析失败。
- `招募` 保留原网页解析作为兜底，API 不可用时仍会尝试返回数据。
- `招募` 输出补充剩余时间、更新时间和装等信息的格式化。

## v0.5.0

### 新增

- `招募` 命令支持 Categories 分类筛选，例如 `招募 陆行鸟 随机任务`。
- `招募` 命令支持指定返回数量，例如 `招募 陆行鸟 随机任务 20`。

### 优化

- `招募` 默认只返回最近 10 条，避免一次发送过长图片。
- 优化招募图片样式，改为更紧凑的条目卡片格式。

## v0.4.0

### 新增

- 新增 `攻略` 命令，支持按副本等级和关键字查询副本攻略，默认图片回复，追加 `文本` 参数时返回文本。
- 新增 `招募` 命令，支持查询指定大区招募板并以图片消息回复。

## v0.3.0

### 新增

- 新增 AstrBot 插件配置项 `默认使用国际服日历`，可在 WebUI 切换默认日历服务器。
- `日历` 命令新增服务器参数，支持 `日历 国服`、`日历 国际服`。

### 优化

- 国服日历使用 Google Calendar 作为主源、iCloud 作为备用源。
- 国际服日历使用 Google Calendar 作为主源、iCloud 作为备用源。

## v0.2.2

- 增加 B 站接口异常日志，便于判断是否遇到 cookies、风控或接口返回异常。
